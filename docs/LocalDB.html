配置参数

Name Type Default Description
expire String "window" optional

"window"：数据随着当前页面标签关闭而消失, "none"：数据会一直存在对应的域内，不随着页面或者浏览器关闭而消失。
encrypt Boolean true optional

true：对存储的数据进行加密操作
proxy String null optional

指定proxy url来进行跨域数据存取，具体请参考@todo Proxy文档
